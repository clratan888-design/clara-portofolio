Florarity: Flower Shop Desktop Application

A desktop flower shop management application with database integration, developed using Java.

About the Project

Florarity is my 3rd-semester project, an application developed for the Business Application Development (BAD) course. This application is a flower shop management system built using Java and integrated with a MySQL database.

The system is designed to serve two main user roles: Admin, who has full authority to manage product data, and Customer, who can register, log in, and view products. For more technical details, please download the project documentation below.
