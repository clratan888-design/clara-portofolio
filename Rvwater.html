RvWater: Application for Local Farmers

LocalFarmer is a project assignment I worked on concerning the database of an application that connects local farmers with consumers.

About the Project

RvWater is a project I completed for the Advanced Information System Analysis and Design (AISAD) course using Visual Paradigm. The LocalFarmer application is designed as a digital platform to connect local farmers with consumers, facilitating the direct sale of fresh agricultural products. In this project, I learned the fundamentals of databases (creating and manipulating data).
